Core Capabilities

Designed for Practical UI Workflows

A focused set of tools that cover the essentials of layout design without unnecessary complexity.

Visual Editing

Create rectangles, text, buttons, images, and shapes. Drag, resize, and organize elements using a layers panel.

Productivity Tools

Undo/redo history, keyboard shortcuts, multi-selection, duplication, and copy/paste for faster iteration.

Precision & Control

Snap-to-grid, alignment tools, and zoom controls ensure predictable and precise positioning.

Export & Persistence

Layouts auto-save locally and can be exported as JSON or standalone HTML files.
